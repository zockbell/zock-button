# zock-button

## Project setup
```
npm i zock-button
```

### 如何使用
* ```main.js```中引入
```
import zock from 'zock-button';
import 'zock-button/zock-button.css';

Vue.use(zock);
```
* 组件中直接使用：
```
<zock-button type="success">按钮</zock-button>
```
